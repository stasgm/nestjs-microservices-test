# Orders microservice

## Installation

```bash
# install the dependencies
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dev

# debug mode
$ pnpm run start:debug

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
